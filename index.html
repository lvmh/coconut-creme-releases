## what you need
⚠ you'll need an IPTV subscription

coconut creme is a player — it doesn't come with any content. to use it you need a subscription from an IPTV provider that supports the Xtream Codes API or M3U playlists.

your provider will give you a server URL, username, and password. enter those on the login screen and you're in.

search for "xtream codes IPTV" to find providers — most offer a 24–72 hour trial before you commit.

## install
01.
download the .dmg
click the button above and open the disk image
02.
drag to applications
move coconut creme.app into your Applications folder
03.
remove quarantine flag
macOS blocks apps distributed outside the App Store — run this once in terminal:
$ xattr -d com.apple.quarantine "/Applications/coconut creme.app"
if macOS still blocks it, go to System Settings → Privacy & Security and click Open Anyway
04.
launch & connect
open the app and enter your IPTV provider URL + credentials
